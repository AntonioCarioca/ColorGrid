# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/)
and this project adheres to [Semantic Versioning](https://semver.org/).

---

## [1.3.0] - 2026-04-28

### Added
- Button "Clear Palette"
- Button "Copy All Colors"
- Export palette to JSON
- Export palette to CSS
- Improved mobile responsiveness

### Improved
- Automatic contrast for text labels based on background color
- Better user feedback when copying colors

### Fixed
- Clipboard error handling (copy failures now show feedback)

---

## [1.2.0] - 2026-04-13

### Added
- Export palette as PNG
- Color editing directly in the grid
- Support for HEX, RGB and RGBA formats

### Improved
- UI layout and interactions

---

## [1.1.0] - 2025-07-01

### Added
- Color copy to clipboard
- Format switch (HEX / RGB / RGBA)

---

## [1.0.0] - 2025-05-12

### Added
- Initial release
- Random color generation (5x4 grid)
- Basic UI layout
